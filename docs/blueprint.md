# **App Name**: JurisPlan LegalTech

## Core Features:

- Content Hub Dinámico (Fallos, Doctrina, FAQ): Visualización y navegación de contenido jurídico clave (fallos, artículos de doctrina, FAQs) almacenado en Firestore, con URLs amigables para SEO.
- Secciones de Problemas Legales SEO-Optimizadas: Páginas dedicadas y estructuradas por tipo de problema (ej. liquidación, rescisión), cada una con explicaciones claras, bloques CTA y un segmento para consulta de caso, fundamentales para la captación orgánica.
- Formulario de Evaluación de Caso: Un formulario completo para capturar datos de clientes potenciales (nombre, WhatsApp, administradora, estado del plan, descripción del caso) y almacenarlos en Firestore para seguimiento.
- Integración Directa con WhatsApp: Un botón flotante de WhatsApp persistente en todas las páginas con un mensaje pre-armado, facilitando la conversión a consultas directas.
- Herramientas SEO Fundamentales: Implementación técnica de SEO (metadatos dinámicos, titles optimizados, URLs limpias, generación de sitemap.xml) para asegurar la visibilidad en buscadores.
- Panel de Administración de Contenido Simple: Una interfaz básica para que los administradores carguen, editen y publiquen fallos, artículos de doctrina y preguntas frecuentes directamente a Firestore, gestionando el contenido sin intervención de desarrollo.
- Asistente AI para Redacción de Resúmenes Legales (tool): Una herramienta impulsada por IA integrada en el panel de administración para asistir en la generación de resúmenes concisos de fallos o en la elaboración inicial de borradores para artículos de doctrina, mejorando la eficiencia en la creación de contenido.

## Style Guidelines:

- Esquema de color claro para evocar profesionalismo y claridad. Color primario: Un azul sofisticado y confiable (#1F5FBD). Color de fondo: Un azul grisáceo muy claro y sutil para la base del sitio (#EBEEF3). Color de acento: Un cian vibrante (#1ACEE5) para resaltar llamadas a la acción y elementos interactivos.
- Tipografía limpia y moderna. Fuente para titulares: 'Space Grotesk' (sans-serif) para una sensación tecnológica y contemporánea. Fuente para cuerpo de texto: 'Inter' (sans-serif) para asegurar legibilidad en textos extensos como doctrina y fallos.
- Iconos limpios, profesionales y temáticamente relevantes para el sector legaltech. Predominio de iconos vectoriales para escalabilidad y rendimiento.
- Diseño grid-based y modular, con uso de cards y bloques visuales para organizar el contenido. Responsive 'mobile-first' asegurando una experiencia óptima en todos los dispositivos. Claridad en la jerarquía visual para guiar al usuario a través de la información jurídica.
- Animaciones sutiles y discretas, como efectos de hover en cards y botones, y transiciones suaves entre secciones para mejorar la experiencia de usuario sin distraer del contenido.
